# 🧿 Oráculo Digital

### 🔮 Herramienta de ayuda para tamear Digimon en Net Driver Digital Monster

Bienvenido a **Oráculo Digital**, una herramienta diseñada para ayudarte a comprender y optimizar el proceso de domesticación (tameo) de Digimon en **Net Driver Digital Monster**. 🚀

## 🚀 Instalación y Uso

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/oraculo-digital.git
   ```
2. **Abre el archivo HTML en tu navegador:**
   - En **Windows**:
     ```bash
     start index.html
     ```
   - En **Mac**:
     ```bash
     open index.html
     ```
   - En **Linux**:
     ```bash
     xdg-open index.html
     ```
